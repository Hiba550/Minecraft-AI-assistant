🤖 Minecraft AI Assistant

Minecraft AI Assistant brings the power of Google Gemini AI directly into your Minecraft world! Get instant help, answers, and even execute commands through natural language — all powered by cutting-edge AI technology. Perfect for new players, builders, and anyone who wants an intelligent companion in their Minecraft adventures.

✨ Key Features

  • 🧠 Smart AI Chat - Ask anything Minecraft-related and get detailed, helpful responses
  • ⚡ Command Interpretation - Tell the AI what you want in plain English, and it executes the right commands
  • 🛡️ Survival Mode Protection - AI respects game modes and won't cheat in survival
  • 🔧 Easy Configuration - Simple in-game setup with professional UI
  • 🚀 Multiple AI Models - Choose from Gemini 2.5 Flash models for optimal performance
  • 🌐 Cross-Platform - Works on Fabric, Forge, and NeoForge

🎮 How to Use

Chat Command

Use /ai <your question> to chat with the AI:

  • /ai How do I make a nether portal?
  • /ai What's the best level to find diamonds?
  • /ai How do I defeat the Ender Dragon?
  • /ai Explain redstone basics

Command Execution

Use /aic <natural language request> to have AI execute commands:

  • /aic Make it night time → AI executes: /time set night
  • /aic Give me 32 oak logs → AI executes: /give @s oak_log 32 (Creative only)
  • /aic Teleport me to spawn → AI executes: /tp @s ~ ~ ~
  • /aic Clear the weather → AI executes: /weather clear

⚠️ In Survival mode, the AI will refuse to give items or XP to maintain fair gameplay!

🔑 Getting Your Gemini API Key

  1. Visit Google AI Studio
  2. Sign in with your Google account
  3. Click "Create API Key"
  4. Select a Google Cloud project (or create a new one)
  5. Copy your API key
  6. In Minecraft, press K to open the config menu
  7. Paste your API key and select your preferred model

💡 The Gemini API has a generous free tier perfect for Minecraft use!

⚙️ Configuration

  • K key opens the config menu
  • Set your Gemini API key
  • Choose from multiple Flash models:
    • Gemini 2.5 Flash Preview (May 20) - Latest and most capable
    • Gemini 2.5 Flash Preview (April 17) - Stable and reliable
    • Gemini 2.0 Flash Experimental - Fastest responses
  • Clean, professional interface with proper spacing and layout

📥 Installation

  1. Download the appropriate version for your mod loader (Fabric/Forge/NeoForge)
  2. Place the .jar file into your mods folder
  3. Launch Minecraft
  4. Press K to configure your API key
  5. Start using /ai and /aic commands!

⚡ Requirements

  • Minecraft: 1.21.5
  • Java: 21 or higher
  • Fabric: Loader 0.16.10+ with Fabric API
  • Forge: 55.0.1+
  • NeoForge: 21.5.4-beta+
  • Internet: Required for AI functionality
